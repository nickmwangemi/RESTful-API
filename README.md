# RESTful API

Clone the repo:

```
$ git clone https://github.com/nickmwangemi/RESTful-API.git
```

Switch to the project directory:

```
$ cd RESTful-API
```

Install project dependencies and fire up the server:

```
$ npm install
$ npm start
# or run with Nodemon
$ npm run dev

# Visit http://localhost:3000
```
